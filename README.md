# 使用gulp打包单页H5项目

用于h5单页项目的基本文件结构以及gulp任务流

# Install

```shell

$ git clone https://github.com/Clearives/gulp_webpack.git
$ npm install

```

# Start

```shell

$ http-server

```


# dev(开发版本)

```shell

$ gulp dev

$ gulp watch   监控代码自动更新开发环境


# publish(正式版本)
> 代码混淆，md5

```shell

$ gulp publish
