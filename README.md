# h5-gulp-webpack

用于h5单页项目的基本文件结构以及gulp任务流

### Install

```shell

$ git clone https://github.com/Clearives/h5-gulp-webpack.git
$ npm install

```

### dev(开发版本)

```shell

$ gulp dev

$ gulp watch   监控代码自动更新开发环境

```

### publish(正式版本)
> 代码混淆，md5

```shell

$ gulp publish

```

### Start

```shell

$ npm start   启动生产环境服务器
$ node app    启动开发环境服务器

```
